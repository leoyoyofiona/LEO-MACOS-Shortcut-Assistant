# LEO-MACOS快捷键助手 0.1.0

首个公开社区预览版。按住一个自定义功能键，即可查看当前应用的快捷键；松开后立即隐藏。

## 中文亮点

- 自动识别当前应用并读取菜单快捷键。
- 支持左/右 Control、Option、Command 和 Fn 触发键。
- 支持简体中文与 English。
- 支持鼠标和触控板滚动浏览完整菜单。
- 自适应半透明水玻璃面板和多个显示位置。

## 中文安装说明

推荐下载 DMG，打开后将应用拖入“应用程序”。首次启动需要允许“辅助功能”和“输入监控”。

当前社区预览版尚未完成 Apple Developer ID 公证。如果 macOS 阻止首次打开，请在 Finder 中右键应用，选择“打开”，再确认一次。不要关闭 Gatekeeper。

---

The first public community preview. Hold a customizable modifier key to reveal shortcuts for the frontmost app, then release to hide.

## Highlights

- Hold a customizable modifier key to reveal shortcuts for the frontmost app.
- Release the key to hide the panel immediately.
- Choose left/right Control, Option, Command, or Fn.
- Switch between Simplified Chinese and English.
- Browse large shortcut sets with mouse or trackpad scrolling.
- Use adaptive panel placement and a translucent macOS-style interface.

## Install

Download the DMG, open it, and drag the app into Applications. ZIP is provided as an alternative.

On first launch, grant Accessibility and Input Monitoring permissions. These are required to read menu shortcut metadata and observe the selected trigger key.

## Current limitation

This community preview is not yet notarized with Apple Developer ID. If macOS blocks the first launch, Control-click the app in Finder, choose **Open**, and confirm. Do not disable Gatekeeper.

See the bilingual README for complete instructions and privacy details.
