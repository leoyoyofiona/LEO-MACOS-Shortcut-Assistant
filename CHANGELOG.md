# Changelog

All notable changes to this project are documented here.

## [0.1.0] - 2026-07-15

### Added

- Frontmost-application detection through macOS Accessibility APIs.
- Hold-to-show and release-to-hide shortcut panel.
- Custom triggers for left/right Control, Option, Command, and Fn.
- Adaptive translucent glass panel with vertical and horizontal scrolling.
- Chinese and English UI with local command translation cache.
- Configurable panel placement.
- Menu bar controls and built-in panel preview.
- Apple Silicon release packaging as ZIP and DMG.
